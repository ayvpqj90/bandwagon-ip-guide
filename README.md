## 换 IP 后还是连不上？排查这几点

换完新 IP，有时候会碰到还是进不去的情况，通常是以下几个原因：

**1. 客户端配置没更新**
代理软件、SSH 连接文件里还写着旧 IP。把配置里的地址改成新的就行。

**2. 旧 IP 仍在 DNS 缓存里**
等几分钟，或者在命令行手动刷新 DNS 缓存再试。

**3. 新 IP 和旧 IP 相差不多时，GFW 规则还没更新**
一般等一段时间就好了，如果一两个小时后还是不行，回复工单让客服确认。

**4. VPS 服务本身的问题**
如果 KiwiVM 面板显示 VPS 是开机状态，但就是 ping 不通，回复工单告知情况，客服会处理。
